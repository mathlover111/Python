# Python 基礎學習筆記與實作記錄

這是一份記錄學習 Python 基礎語法的筆記，包含環境建置、基礎運算以及流程控制的練習題。

---

## 📁 Step 01：認識 Python
學習如何安裝 Python 環境、操作 VS Code 編輯器，以及印出第一行程式碼。

* **`01_hello.py`**
  * **功能**：印出第一個 "Hello World" 文字。
  * **重點**：確認 Python 開發環境與執行功能正常。

---

## 📁 Step 02：基礎語法
變數宣告、使用者輸入（`input`）、型態轉換，以及基礎數學運算。

* **`02_f-string.py`**
  * **功能**：輸入名字，並輸出個人化的問候語。
  * **重點**：學習 `input()` 取得使用者資料，並透過 `f-string` 格式化輸出。

* **`03_currency_converter.py`**
  * **功能**：輸入台幣金額，依匯率 32 換算成美金。
  * **重點**：學習 `float()` 型態轉換，避免文字與數字相除報錯。

* **`04_square.py`**
  * **功能**：輸入邊長，自動算出正方形的周長與面積（只印出數字）。
  * **重點**：將同一個輸入數值同時進行乘法與平方運算。

* **`05_temp_converter.py`**
  * **功能**：將攝氏溫度轉換為華氏溫度。
  * **重點**：實作數學公式計算：`celsius * 1.8 + 32`。

* **`06_string_multiplier.py`**
  * **功能**：輸入任意單字與次數，將單字重複印出。
  * **重點**：學習 `int()` 轉換，並利用 Python 特有的「字串 * 數字」特性。

---

## 📁 Step 03：流程控制
本階段進入邏輯核心，練習條件判斷式（`if`、`elif`、`else`）、邏輯運算子（`and`、`or`、`!=`），以及 `for` 與 `while` 迴圈。

* **`01_odd_even.py`**
  * **功能**：輸入一個整數，判斷是奇數還是偶數。
  * **重點**：使用 `% 2` 取餘數，並認識 `==` 比較運算子。

* **`02_score_level.py`**
  * **功能**：輸入分數，自動判斷並顯示 A、B、C 或不及格。
  * **重點**：學習多重條件判斷 `elif` 的由上到下執行順序。

* **`03_ticket_price.py`**
  * **功能**：判斷是否符合「免票」或「全票 200 元」資格。
  * **重點**：學習邏輯運算子 `or`（多條件中只要一者滿足即可）。

* **`04_leap_year.py`**
  * **功能**：輸入年份，自動計算並判斷是閏年還是平年。
  * **重點**：綜合運用 `and`、`or` 以及 `!=`（不等於）處理複雜邏輯。

* **`05_print_numbers.py`**
  * **功能**：自動印出 1 到 10 的數字。
  * **重點**：掌握 `for` 迴圈與 `range()` 區間「包頭不包尾」的機制。

* **`06_sum_1_to_100.py`**
  * **功能**：自動算出 1 加到 100 的總和。
  * **重點**：建立累加變數，並練習 `total += i` 簡寫。

* **`07_nine_nine.py`**
  * **功能**：自動印出完整的 1x1 到 9x9 乘法表。
  * **重點**：實作雙重迴圈（Nested Loop），理解外圈與內圈的連動邏輯。

* **`08_guess_number.py`**
  * **功能**：讓使用者一直猜數字，猜對才結束程式。
  * **重點**：學習 `while` 迴圈，用於「不確定重複次數」的場景。

* **`09_break_continue.py`**
  * **功能**：印出 1 到 10，但跳過 4、並在遇到 8 時提早結束。
  * **重點**：掌握 `continue`（跳過本輪）與 `break`（中斷迴圈）指令。

* **`10_password_check.py`**
  * **功能**：模擬 ATM 輸入密碼，最多錯 3 次即鎖定。
  * **重點**：綜合運用迴圈次數限制、密碼判斷與 `break` 提早結束。
